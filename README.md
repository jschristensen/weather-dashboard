# European Capitals Weather Dashboard

A single-page **React + Vite + TypeScript** dashboard showing the current weather in
the 10 most populous European capitals, using the free [Open-Meteo](https://open-meteo.com)
API (no API key required). Data is fetched once on page load.

See [`specs/weather-dashboard.md`](specs/weather-dashboard.md) for the full spec.

## Run

```bash
npm install
npm run dev      # start dev server
npm run build    # type-check + production build
```
